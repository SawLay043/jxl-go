# Tech Stack: jxl-go

## Language
- **Go (v1.26.0):** The primary programming language for the project, providing a balance of performance, safety, and ease of development.

## Frameworks and Libraries
- **Standard Library:** Extensive use of Go's built-in packages for fundamental tasks like I/O, concurrency, and image handling.
- **Logrus:** Utilized for structured and descriptive logging throughout the library.
- **Testify:** Employed for more robust and readable unit testing.
- **Profile:** Used for application profiling to identify and address performance bottlenecks.
- **Go Exp:** Leveraging experimental Go features to explore potentially beneficial future additions to the language.

## Development and Deployment Tools
- **Makefile:** Used for managing development tasks like building, testing, and running examples.
- **GitHub Actions:** Configured for continuous integration to automate testing and code verification on every push.
